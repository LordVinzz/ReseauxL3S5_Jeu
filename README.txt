Les serveurs ne s'arretent qu'avec CTRL+C, il n'est pas indiqué
dans le sujet si on doit les arreter quand tous les joueurs gagnent
donc je n'ai pas integre une telle fonctionnalite.

Il n'y a pas de verification des donnees entrees car ce n'est pas indique
dans le sujet, en soit il suffirait de vérifier si le message contient
autre chose que des chiffres et des espaces et \0 et redemander de les entrer
si ce n'est pas le cas.
